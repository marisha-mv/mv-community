Key Insight

Completion, Not Connection

People don't want to "connect about quests." So don't make them. Focus on what actually drives completion.

The Team's Observation

"People don't want to connect about quests ‚Äî but we need to make sure they finish."

This is important. If true, traditional community features (discussion forums, study groups, "share your experience") might create FRICTION, not motivation.

The goal isn't connection. It's completion. Connection is only valuable if it serves completion.
